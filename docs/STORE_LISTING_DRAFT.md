## App Name

PitWise

## Subtitle / Short Description

OBD2 code help before the mechanic.

## Full Description

PitWise helps drivers understand OBD2 diagnostic trouble codes in plain English before talking to a mechanic.

Enter a code manually or connect a compatible Bluetooth OBD2 adapter to read stored codes. PitWise explains likely causes, urgency, drive-safety guidance, common repair paths, estimated repair-cost ranges, proof to request, and questions to ask before approving work.

PitWise is built for preparation, not guesswork. It uses cautious language and encourages professional inspection when safety-sensitive symptoms are present.

Features:

- Plain-English OBD2 code explanations.
- Manual code entry.
- Bluetooth OBD2 adapter scanning for compatible BLE adapters.
- Saved vehicle context.
- Scan history.
- Mechanic-prep checklist.
- Privacy controls and in-app data deletion.

Important: PitWise is informational only. It does not provide a confirmed diagnosis, replace a qualified mechanic, or determine whether a vehicle is safe to drive.

## Reviewer Notes

PitWise can be tested without Bluetooth by using manual OBD2 code entry. Suggested codes: P0302, P0420, P0171.

Bluetooth scanning requires a compatible BLE OBD2/ELM327-style adapter connected to a vehicle OBD2 port. The app requests Bluetooth permissions to discover and connect to the adapter. Android may request location permission for Bluetooth scanning depending on OS behavior.

No login is required. The app uses a random app-install ID to isolate saved vehicle and scan history. Data deletion is available in Settings > Delete My Data.
